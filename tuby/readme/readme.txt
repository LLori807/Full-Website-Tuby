Changing The Featured Videos
- Open for editing index.html
- Find the Featured section
- Then find the following line of code:
<iframe src="https://www.youtube.com/embed/8b8NJZi_rcQ" allowfullscreen></iframe>

- And replace the video code 8b8NJZi_rcQ with your own video code from your youtube link
- Repeat for the other slides


-----------------------------------------------------


Changing The Latest Videos
- Open for editing index.html
- Find the Latest section
- Then find the following line of code:
<a class="popup-youtube" href="https://www.youtube.com/watch?v=zqkwVAczbkA" data-effect="fadeIn"><img class="img-fluid" src="images/latest-videos-1.jpg" alt="alternative"></a>

- Here replace the video code zqkwVAczbkA with your own video code from your youtube link
- Also you can update the video image by replacing the image images/latest-videos-1.jpg
- Repeat for all videos
